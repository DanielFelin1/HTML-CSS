Colocando vídeos no seu site

Para inserir vídeo em nosso site podemos utilizar a tag <video>, caso o arquivo esteja hospedado
em nosso próprio servidor.

Vamos analisar os principais atributos da tag <video>:

  • widht vai indicar a largura que o video vai ter na tela.

  • poster configura uma imagem que vai aparecer como uma capa, enquanto o visitante
    não aperta o play para reproduzzir o video.

  • controls vai configurar se os controles do vídeo vão aparecer na parte inferior da mídia.
    Por padrão, os controles não aparecerão, mas basta colocar a palavra controls na tag video.

Oa formatos suportados pelos navegadores são MPEG, WEBM e OGG,
mas os dois primeiros são os que possuem maior compatibilidade com os navegadores atualmente.
