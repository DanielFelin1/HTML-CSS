Vamos criar um link

Hyperlinks:

São um dos conceitos mais antigos na linguagem HTML. Eles permitem que você ligue um ponto a outro na World Wide Web.
Para criar um Hyperlink, devemos criar âncoras atravez da tag <a>. O principal atributo dessa tag é o href, que cria uma referencia hipertexto.

Usando hreflang:

O hreflang permite indicar qual é o idioma principal do site par onde o link está desviando o fluxo de navegação. permite que o visitante possa optar pela tradução do idioma do site.

Usando o atributo target

Para poder controlar onde o site de destino vai abrir, podemos usar o atributo target, que suporta os segiontes valores:

_blank vai abrir o link em uma nova janela em branco.
_self vai abrir o link na janela ou frame atual (padrao).
_top vai desfazer todos os frames e abrir o destino no navegador completo.
_parent similar ao uso do _top em uma referencia a janela mae.
nome-do-frame caso esteja usando frames, indicar o nome da janela a abrir.

Esse link é seu ou dos outros?

Existe um recurso bem interessante para links que é indicar qual é a natureza do destino usando o atributo rel. Esse atributo aceita varios valores, entre eles:

next - Indica que o link é a proxima parte do documento atual
prev - Indica que o link é para a parte anterior do documento atual
author - Indica que é um link para o site do autor do artigo atual.
external - Indica que é um link para outro site que nao faz parte do site atual.
nofollow - indica que é um link para um site não endossado, como um link pago.

E para fazer Download?

Basta fazer o link diretamente para o arquivo que deseja efetuar o download e adicionar o atributo download com o valor configurado para o nome do arquivo a ser baixado, e o atributo type para indicar ao navegador que tipo de arquivo esta sendo baixado.
