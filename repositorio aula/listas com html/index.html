Listas com HTML

Listas Ordenadas

A HTML chama de ordered lists todas as listas em que a ordem do itens é de extrema importancia.

Para criar uma ordered list usamos a tag <ol> para delimitar a lista, e <li> para identificar cada idem da lista.
Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5
  6. Item 6

A tag <ol> possui um parametro type onde configuramos o tipo de marcador da lista.
Exemplo:

1 - Valor padrão. Listas enumeradas: 1, 2, 3, 4, 5...
A - Listas em ordem alfabetica com letras maiusculas: A, B, C, D, E...
a - Listas em ordem alfabetica minusculas: a, b, c, d, e...
I - Algarismos romanos: I, II, III, IV, V...
i - Algarismos romanos em minusculo: i, ii, iii, iv, v...

Podemos indicar o inicio da contagem usando o parametro start.
Exemplo:

<ol> type="i" start = "5"></ol>

Nesse exemplo a lista será gerada apartir do item 5.

Listas não Ordenadas

Nas unordered lists a ordem dos intens não irá importar.

Para criar uma unordered list, usaremos a tag <ul> para delimitar a lista e a tag <li> para criar os seus itens internos.
Exemplo:

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

Misturando as coisas

Podemos também, criar listas mistas configurando listas dentro de outras listas. Exemplo:

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
  3. Item 3
  4. Item 4
  5. Item 5

Lista de definições

É como se fosse um dicionário, temos os termos e suas definições.

Toda lista de definição esta dentro de uma tag <dl> </dl> (definition list).
Cada termo é um <dt> (definition term).
Cada descrição é uma <dd> (definition description).
Exemplo:

HTML
Linguagem de marcação utilizada para criar o conteudo de sites.
CSS
Linguagem de marcação para a especificação de estilos em sites.
Jvascript
Linguagem de programação para criar interatividades em sites.
